Projeto desenvolvido em dupla, produzido presencialmente em momentos vagos na faculdade.



Uso de GenAI: 
  Utilizamos IA para nos informar quais as formas de ler dados internos do pc, para exibir hora, processos, taxa de cpu, memória ram e cpu.
  A GenAI contribiu para tirar duvidas do que seria possivel ou não fazer.




Sobre o Script:
  Nosso script foi planejado para servir como um monitor de sistemas, que ajuda em acompanhar o uso de: Memória RAM, CPU, Disco, e top 3 
processos que mais estão utilizando memória RAM. O executável é atualizado a cada 1 segundo, fazendo com que o usuário tenha um acompanhamento
melhor, mas não extremamente simultâneo.
  Outra funcionalidade é o alerta, sempre que acima de tal valor, aparece mensagem de alerta para o usuário.
  RAM: Acima de 85%;
  CPU: Acima de 90%;
  Disco: Acimad e 95%.




Objetivo:
  Para o programador, é importante monitorar o uso de seus componentes, para evitar problemas talvez até graves. Sabendo que 
memoria RAM e processador é fundamental para a produtividade do profissional, nosso objetivo foi facilitar o programador a acompanhar 
o que está acontecendo com o uso de sua máquina.




Instruções de uso:
  o programa é muito simples de utilizar, basta baixar o script, executá-lo no terminal com <./monitor.sh> e logo em seguida 
  o terminal exibe as informações prometidas.
